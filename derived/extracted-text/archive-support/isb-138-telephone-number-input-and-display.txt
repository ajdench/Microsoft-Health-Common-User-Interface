Standard ISB 1508

 Number
 ISB 1508

 Title
 Common User Interface - Telephone Number Input and Display

 Description

This user interface standard defines the valid values for input and display of telephone numbers, definition of telephone number elements and values, how items of information are to be labelled, the layout and structure of the control and input field sizes.

 Applies To

IT systems that capture or display telephone numbers.
