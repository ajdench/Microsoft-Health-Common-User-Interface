Standard ISB 1500

 Number
 ISB 1500

 Title
 Common User Interface - Address Input and Display

 Description

This user interface standard specifies the rules for the display of addresses, input of address data elements, input of UK addresses, the UK address finder and input of non-UK addresses.

 Applies To

IT systems that capture or display address data.
