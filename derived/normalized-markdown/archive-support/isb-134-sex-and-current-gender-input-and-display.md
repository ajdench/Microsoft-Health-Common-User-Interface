 Number
 ISB 1507

 Title
 Common User Interface - Sex and Current Gender Input and Display

 Description

This user interface standard defines the valid terms and values for sex and current gender, labelling of information, layout and structure of the control and input field sizes.

 Applies To

IT systems that capture or display sex and/or gender data.
