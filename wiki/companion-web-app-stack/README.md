# Companion Web App Stack

## Purpose
This knowledge space is a companion to the primary Microsoft Health CUI wiki.
It is not part of the recovered historical Health CUI corpus.

The purpose is to capture modern implementation guidance that is informed by
the Health CUI wiki, but which depends on current web platform, browser,
offline-first, and UI-library research.

## Boundary
Use this area for:
- modern web application stack choices
- offline-first and local-first architecture notes
- PWA, Microsoft Edge, WebView2, and storage strategy notes
- modern UI library assessment
- implementation recommendations derived from, but not asserted as, Health CUI source material

Do not use this area for:
- newly acquired historical Health CUI source material
- raw source provenance
- primary corpus source notes
- formal claims about historical Health CUI that are not already supported elsewhere in the wiki
